Bienvenue sur l'application de gestion de recettes

Partenariat entre Adepro et le Lycée Vinci

Cette application est le fruit d'un partenariat entre Adepro et le Lycée Vinci. L'objectif est de fournir une solution innovante pour la gestion des recettes culinaires.

Utilité de l'application

Cette application permet de :

  • Créer, modifier et supprimer des recettes.
  • Gérer les ingrédients nécessaires pour chaque recette.
  • Visualiser les recettes de manière intuitive.

Public cible

Cette application est destinée aux :

  • Élèves du Lycée Vinci pour apprendre la gestion de projets.
  • Professionnels de la restauration pour organiser leurs recettes.
  • Amateurs de cuisine pour partager leurs créations.
